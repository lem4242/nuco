---
description: Print a friendly hello-world greeting from the plugin.
---

Greet the user with a cheerful hello-world message.

If `$ARGUMENTS` is non-empty, treat it as the name to greet and say
"Hello, $ARGUMENTS!". Otherwise say "Hello, world!".

Then add a single short line noting that this greeting came from the
`hello-world` example plugin's `/hello` command, **version 0.2.0**.
